# 🎓 Student Management System PRO

An advanced CLI-based Student Management System built using Python.
This project demonstrates Object-Oriented Programming (OOP), file handling, and data management.

---

## 🚀 Features

* ➕ Add Students
* 📋 List Students
* 🔄 Update Student Details
* 🗑️ Delete Students
* 🔍 Search (Name / ID)
* 🔢 Sorting (Name / Grade)
* 📊 Statistics (Average, Topper)
* 📦 Backup System (JSON)
* 📁 Export Data to CSV
* 🔐 Admin Login System

---

## 🛠 Tech Stack

* Python
* JSON (data storage)
* CSV (export feature)
* OOP (Classes & Objects)

---

## ▶️ How to Run

```bash
python3 student_manager.py
```
